## Key Technical Differentiators

- **Coordinate-based browsing** (vs CSS selectors): Gemini Computer Use returns (x,y) on normalized 1000x1000 grid — works on any website
- **24kHz audio output**: Higher quality voice than standard 16kHz
- **Fully async**: Pure asyncio with Event synchronization (no threading)
- **Native WebSocket**: No Socket.IO dependency, smaller footprint
- **GCP ADC**: No API keys in code — uses Application Default Credentials
- **13 action types**: click, type, scroll, hover, key, drag, select, wait, navigate, go_back, go_forward, done, read_page
